# Age Eligibility Checker

## 📌 Project Description
This project checks a user's age and displays whether they are eligible for school, voting, driving, or senior citizen benefits.

---

## 🚀 Features
- Age input validation
- Eligibility check using conditions
- Colorful result messages
- Responsive modern UI
- Glassmorphism design

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

---

## 📚 Concepts Used
- If-else conditions
- Functions
- DOM manipulation
- Input handling
- UI styling

---

## 🎯 How to Run
1. Download the project
2. Open `index.html` in browser
3. Enter age and click button

---

## 🌟 Output Categories
- 👶 Child (below 5)
- 🎓 Student (5–17)
- ✅ Eligible adult (18–59)
- 👴 Senior citizen (60+)
